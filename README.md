Regions: Asia, Europe
Average Latency: 24ms

- url: `dhifonts.b-cdn.net`
- usage: `dhifonts.b-cdn.net/<font_name>`
- example: `dhifonts.b-cdn.net/mvtyper.ttf`

#### Code Example?
